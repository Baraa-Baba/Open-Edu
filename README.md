# Open-Edu

Open-Edu is an educational platform built to organize and share study materials for Lebanese Baccalaureate students (Grade 12).

## About

This project is focused on helping students access their school content in one place, with clear structure and an easy browsing experience.

### Target Audience

Lebanese Baccalaureate students (Grade 12) and educators supporting the official Lebanese curriculum.

## Features

- Organized folders and files for learning materials
- Subject-based structure for easier navigation
- Admin and authentication flow for protected content management
- Responsive interface for desktop and mobile access

## Tech Stack

- React
- Vite
- Firebase
- CSS

## Getting Started

### Prerequisites

- Node.js 18+ (recommended)
- npm

### Installation

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open the local URL shown in your terminal.

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build locally

## Project Structure (High-Level)

- `src/components` - Reusable UI components
- `src/routes` - Main application pages and route-level views
- `src/context` - Shared app context (authentication and global state)
- `src/Subjects` - Subject data and material mapping
- `public` - Static assets

## Purpose

The purpose of Open-Edu is to support Lebanese Baccalaureate preparation by making Grade 12 educational resources easier to access, organize, and maintain.
